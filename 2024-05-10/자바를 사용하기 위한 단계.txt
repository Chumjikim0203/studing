1. Java란 무엇인가?

- Java는 소스코드로 이루어진 프로그래밍을 위한 언어들을 담은 변수와 함수를 하나의 "객체(클래스)" 로 묶고 그 클래스들을 "패키지"로 묶어 관리하여 사용하는

"객체지향 언어" 이다. 이는 특정 객체단위로 관리를 하여 필요한 객체를 사용하려는 상황에 맞게 호출하여 사용하게 하고, 객체 기반으로 사용하게 하는 언어이다.

또한 Java의 강점은 메모리를 자동으로 관리해주며, 무료 라이브러리 또한 다양하고 다채롭게 사용할 수 있다.


2. JDK(Java Development Kit)란 무엇인가?

- Java는 프로그래밍 언어이며, 프로그래밍 언어는 사람이 작성하는 소스코드이므로 기본적으로 컴퓨터가 즉시 이를 이해 할 수 없다.

이는 컴퓨터가 이해하는 언어인 "기계어"로 번역이 이루어지지 않았기 때문이며, 이 해석과정을 "컴파일(Compile)"이라고 하며, JDK는 Java로 작성된

소스코드 즉, 프로그래밍 언어를 컴퓨터가 이해할 수 있도록 내용을 기계어로 번역해주는 "컴파일러(Compiler)"의 역할을 담당한다.

이 외에도 다양한 기능을 제공하여 Java를 사용하여 개발을 하는데 필요로 하는 환경 및 도구를 제공해준다.


3. Java를 사용하기 위한 JDK 설치 과정

- Java를 사용하기 위해서는 우선 컴퓨터가 작성한 코드를 이해할 수 있도록 번역기능을 제공하는 JDK를 설치해주어야 한다.

이 과정은 다음과 같다.

1) https://www.oracle.com 에 접속하여 Products를 클릭

2) Hardware and Software의 Java탭을 클릭

3) 아래에 있는 Oracle OpenJDK를 클릭

4) 이후 사용에 필요한 운영체제에 맞는 JDK 버전을 다운로드

5) 다운로드한 압축파일을 압축해체 한 이후 C드라이브 - Program Files - Java 폴더 안에 붙여넣기 혹은 이동

6) 윈도우 검색을 통해 "시스템 환경 변수 편집"을 실행하여 하단의 "환경 변수"를 클릭

7) 아래쪽 "시스템 변수" 박스 안에 있는 "Path"를 더블클릭 혹은 "새로 만들기"를 사용하여 JDK의 bin 폴더가 설치되어 있는 경로를 붙여넣어서 경로 설정

8) 이후 "명령 프롬포트(CMD)"를 실행하고 "java -version"을 입력하여 Java가 정상적으로 설치 되어있는지 확인.


4. 개발을 위한 IDE(Intergrated Development Environment) 설치

- 개발자가 각종 프로그래밍 언어(Java, Python, C++등)를 사용하여 코드를 작성하기 위해 필요한 툴이 바로 "IDE"이다.

이는 코드의 흐름을 파악하여 오류나 오타를 잡아주는 역할도 하며, 코드를 기반으로 작성된 프로그램을 임시로 실행하여 정상 작동되는지 확인도 해주며,

디버깅, 프로젝트 생성, 자동 코드 완성등의 다양한 기능을 제공하여 개발자가 좀 더 코드를 작성하고 프로젝트를 작성하기 쉽게 만들어주는 도구이다.

IDE의 종류는 다양하며 대표적으로 "Eclipse", "Visual Studio Code", "InteliyJ"등이 존재한다.

다음은 가장 대중적으로 사용되는 Eclipse의 설치 과정이다.


1) Eclipse 다운로드를 검색하여 https://www.eclipse.org로 접속

2) Download를 클릭하고 사용자의 PC 운영체제에 맞는 Eclipse 설치파일을 다운로드

3) 이후 사용하려는 IDE를 선택하여 설치
